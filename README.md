
# Docker Kali Linux
Kali linux with docker prepared for linux operating system.


## Installation
```sh
git clone https://github.com/erdinccurebal/docker-kali-linux.git
sudo chmod -R 777 docker-kali-linux/
cd docker-kali-linux
bash install.sh
```

## Start
```
bash start.sh
```

## Stop
```
bash stop.sh
```

## Uninstallation
```
bash uninstall.sh
```
